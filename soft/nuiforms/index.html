nuiforms

Go library for building desktop UIs: controls, layout, and interaction on top of the nui windowing layer.

  • Depends on nui for native windows, input, and framebuffer drawing.
  • Targets Go applications that need a graphical UI without embedding a separate browser or heavy toolkit.
