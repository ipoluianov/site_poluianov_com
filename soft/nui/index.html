nui

Low-level Go API for desktop GUI shells: native windows, input dispatch, and raster output. Higher-level widgets live in nuiforms.

  • Window creation and window lifecycle events.
  • Keyboard and mouse input events.
  • Drawing arbitrary pixel buffers into the window client area.
